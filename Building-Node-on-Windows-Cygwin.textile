 3) Install devel -> make
 4) Install python -> python
 5) Install devel -> git
ii) Clone the repo
 1) git clone git://github.com/ry/node.git
iii) run configure in the node folder created above
 1) ./configure
iv) run make
 v) run make test (There will be some test failures in windows)

h3. Know issues:

- domain name resolution : Cygwin internally uses Windows for DNS queries. Node.js uses the c-ares library that relies on /etc/resolv.conf. Cygwin ships with an empty /etc/resolv.conf. In order to enabled networking add a domain name server to the file:

@# add the IP of a valid name server@
@nameserver 11.22.33.44@
